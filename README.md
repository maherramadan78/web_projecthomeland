# web_projecthomeland
# De patria a patria

## Descripción
Este proyecto es una galería de arte interactiva que presenta historias y fotografías de diferentes lugares del mundo. El sitio web muestra un diseño responsivo que se adapta a diferentes dispositivos (desktop, tablet y mobile) y cuenta la historia de profesionales de TripleTen de diversos rincones del mundo.

## Funcionalidades
* Diseño web responsivo para desktop (1280px), tablet (768px) y mobile (320px)
* Galería de imágenes interactiva
* Navegación fluida entre secciones
* Optimización para diferentes dispositivos

## Tecnologías utilizadas
* HTML5
* CSS3
* Flexbox
* Grid CSS
* Metodología BEM
* Media queries para diseño responsivo
